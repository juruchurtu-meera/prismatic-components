Fluent Commerce offers a [GraphQL-based](https://graphql.org/) API.
This gives you flexibility and control over what data you send to Fluent, and what data you request back.

While this component offers some actions that wrap common GraphQL queries and mutations, there are many more queries and mutations for which there aren't dedicated actions.
If you'd like to run a query or mutation that is not represented by an action, please use the [Generic GraphQL Request](#genericrequest) action.
