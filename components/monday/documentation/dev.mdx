Monday offers a [GraphQL-based](https://graphql.org/) API and was built using the [Monday API Reference Version 2026-01](https://developer.monday.com/api-reference/reference/about-the-api-reference).
This provides flexibility and control over what data is sent to Monday, and what data is requested back.

While this component offers some actions that wrap common GraphQL queries and mutations, there are many more queries and mutations for which there aren't dedicated actions.
To run a query or mutation that is not represented by an action, use the [Generic GraphQL Request](#genericrequest) action.
