If Odoo is expected to return lots of records, the integration can leverage [pagination](integrations/common-patterns/loop-over-paginated-api.mdx) to process a small set of records at a time.
See Odoo's [External JSON-2 API reference](https://www.odoo.com/documentation/19.0/developer/reference/external_api.html) for details on the `limit` and `offset` parameters used for pagination.
