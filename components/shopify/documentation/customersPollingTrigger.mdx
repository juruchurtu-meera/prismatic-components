Checks for new and updated customers in a Shopify store on a configured schedule.

#### How It Works

The trigger operates by:
1. Running on the configured schedule (e.g., every 5 minutes)
2. Querying the [Shopify GraphQL API](https://shopify.dev/docs/api/admin-graphql) for customers modified since the last poll using the `updated_at` filter
3. Categorizing customers as "created" or "updated" based on their `createdAt` timestamp compared to the last poll time
4. Maintaining state automatically to track the last poll time
5. Returning all customers on the first run (when no previous poll state exists)

#### Configuration

The trigger requires only a Shopify connection. Schedule configuration is handled by the platform.
