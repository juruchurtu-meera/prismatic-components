This action wraps QuickBooks' [Query a Report](https://developer.intuit.com/app/developer/qbo/docs/api/accounting/report-entities/vendorexpenses#query-a-report) API endpoint.
Query parameters can be added to filter results by `customer`, `vendor`, `department`, `accounting_method`, or `class`.
Keys `summarize_column_by` and `sort_order` determine how the result should be structured.
Specify a `date_macro`, or a combination of a `start_date` and `end_date` (both need to be specified) to get expenses from a particular date range.
