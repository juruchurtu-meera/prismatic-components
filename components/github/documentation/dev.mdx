The [GitHub REST API](https://docs.github.com/en/rest) is extensive, with almost a thousand unique endpoints.
This component has actions that wrap some of those endpoints that are commonly used in integrations.
Please [contact us](https://prismatic.io/contact/) if there are other GitHub endpoints that you would find helpful wrapped in an action.
Alternatively, you can reach for the [Raw Request](#rawrequest) action to interact with any GitHub endpoint.
